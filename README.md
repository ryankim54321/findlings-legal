# findlings-legal

Hosts the Privacy Policy and Terms of Service for the Findlings iOS app via GitHub Pages.

- Live: https://ryankim54321.github.io/findlings-legal/
- Privacy: https://ryankim54321.github.io/findlings-legal/privacy.html
- Terms: https://ryankim54321.github.io/findlings-legal/terms.html

## ⚠️ Before launch

The `privacy.html` and `terms.html` files in this repo are **starter templates** reflecting what the Findlings iOS app currently does. Before production launch:

1. Have a legal advisor review both documents.
2. Replace every `TODO` placeholder (effective date, jurisdiction).
3. Update the "Effective date" in both files to the actual launch date.
4. If you add new data practices (analytics, ads, etc.), update the Privacy Policy accordingly.

## Making updates

Edit the HTML files directly on GitHub or via a local clone, commit to `main`, and GitHub Pages will redeploy within a minute.
